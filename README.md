# 🤝 SOV Bot — School of Volunteers

Telegram-бот для управления волонтёрской организацией SOV.

---

## 🚀 Быстрый старт

### Шаг 1 — Установи Python

Скачай Python 3.11+ с [python.org](https://python.org) и установи.

### Шаг 2 — Создай бота в Telegram

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй **токен** — он выглядит примерно так: `7123456789:AAFxxx...`

### Шаг 3 — Узнай свой Telegram ID

Напиши боту **@userinfobot** — он покажет твой числовой ID.

### Шаг 4 — Заполни config.py

Открой файл `config.py` и вставь свои данные:

```python
BOT_TOKEN = "7123456789:AAFxxx..."   # токен от BotFather
ADMIN_IDS = [123456789]               # твой Telegram ID
```

### Шаг 5 — Установи зависимости

Открой терминал в папке с ботом и выполни:

```bash
pip install -r requirements.txt
```

### Шаг 6 — Запусти бота

```bash
python main.py
```

Бот запустится. В терминале увидишь:
```
База данных готова ✓
Планировщик запущен ✓
Бот SOV запущен 🚀
```

## 📁 Структура проекта

```
sov_bot/
├── main.py          — точка входа, запуск бота
├── config.py        — ТОКЕН и список ADMIN_IDS (заполни!)
├── database.py      — вся работа с базой данных SQLite
├── keyboards.py     — все кнопки и меню
├── scheduler.py     — ежемесячные уведомления топ-3
├── requirements.txt — зависимости
├── handlers/
│   ├── user.py      — команды для участников
│   └── admin.py     — команды для администратора
└── sov.db           — база данных (создаётся автоматически)
```

## 🤖 Как работает автоподбор

**Алгоритм:**
1. Берёт все заявки на ивент
2. Сначала заполняет квоту парней (если указана)
3. Потом квоту девушек (если указана)
4. Оставшиеся места — лучшие по рейтингу + опыту
5. При равном рейтинге → приоритет по опыту (кол-во ивентов)

**После автоподбора:**
- Статусы в базе обновляются автоматически
- Выбранные участники получают уведомление в Telegram
- Список приходит тебе в чат

## 🗄️ Просмотр базы данных

Скачай **DB Browser for SQLite** (бесплатно, Windows/Mac/Linux):
👉 https://sqlitebrowser.org/

Открой файл `sov.db` — увидишь все таблицы в виде таблицы как в Excel:
- `users` — все участники
- `events` — все ивенты
- `applications` — кто на что записался
- `ratings` — все оценки

## ⚙️ Запуск в фоне (чтобы бот работал без открытого терминала)

### Windows — bat файл
Создай файл `start.bat`:
```bat
@echo off
python main.py
pause
```

### Linux/Mac — screen
```bash
screen -S sovbot
python main.py
# Нажми Ctrl+A затем D чтобы свернуть
```

### Для постоянного хостинга
Рекомендуется разместить на VPS (например, [DigitalOcean](https://digitalocean.com) или [Hetzner](https://hetzner.com)) и настроить systemd-сервис.

## ❓ Частые вопросы

**Бот не запускается — ошибка токена**
→ Проверь что токен в `config.py` указан без лишних пробелов и кавычек правильно расставлены.

**Команда /admin не работает**
→ Проверь что твой Telegram ID в списке `ADMIN_IDS` в `config.py`.

**Как добавить второго администратора?**
→ Добавь его ID в список: `ADMIN_IDS = [123456789, 987654321]`

**Как сделать резервную копию данных?**
→ Просто скопируй файл `sov.db` в надёжное место. Это вся база данных.
